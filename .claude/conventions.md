# Conventions

## Conventions

- Architecture style: modular
- Every directory under src/ MUST have an index.ts barrel export

## File Naming
- Components: PascalCase (UserProfile.tsx)
- Hooks: camelCase with use prefix (useAuth.ts)
- Utils/Services: camelCase (formatDate.ts)
- Tests: same name + .test suffix (UserProfile.test.tsx)
